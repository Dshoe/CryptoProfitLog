# Coinbase Profit
A crypto currency purchase and profit tracker.

## Run this project

### Ensure Ionic and Cordova are installed
```bash
$ npm install -g ionic cordova
```

### Add CORS to CouchDB
```bash
$ npm install -g add-cors-to-couchdb
$ add-cors-to-couchdb
```

### Run in the browser
```bash
$ ionic serve
```
